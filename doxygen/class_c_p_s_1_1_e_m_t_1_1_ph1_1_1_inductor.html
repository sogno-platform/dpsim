Detailed Description

Inductor.

The inductor is represented by a DC equivalent circuit which corresponds to one iteration of the trapezoidal integration method. The equivalent DC circuit is a resistance in paralel with a current source. The resistance is constant for a defined time step and system frequency and the current source changes for each iteration.
