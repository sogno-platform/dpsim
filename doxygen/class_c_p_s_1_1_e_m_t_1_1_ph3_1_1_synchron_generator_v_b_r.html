Detailed Description

Synchronous generator model If parInPerUnit is not set, the parameters have to be given with their respective stator or rotor referred values. The calculation to per unit is performed in the initialization. The case where parInPerUnit is not set will be implemented later. parameter names include underscores and typical variables names found in literature instead of descriptive names in order to shorten formulas and increase the readability
